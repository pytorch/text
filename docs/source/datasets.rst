torchtext.datasets
==================

.. currentmodule:: torchtext.datasets

General use cases are as follows: ::


    # import datasets
    from torchtext.datasets import IMDB

    train_iter = IMDB(split='train')

    def tokenize(label, line):
        return line.split()
     
    tokens = []
    for line in train_iter:
        tokens += tokenize(line)
